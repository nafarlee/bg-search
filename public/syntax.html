BG Search

Simple Terms

Tag Aliases Description Example
name Matches a part of the games primary name name:catan

Matches any game with catan somewhere in it's name

art Matches a part of the name of any of the game's artists
art:jakub

Matches any game with art done by anyone named jakub

category Matches a part of the name of any of the game's categories
category:act

Matches any game in the Abstract Strategy, Industry / Manufacturing, or Action / Dexterity categories

desc Matches part of the games description
desc:flip

Matches any game that includes the word flip in it's description

family Matches a part of the name of any of the game's families
family:dominion

Matches any game in the Dominion family

mechanic Matches a part of the name of any of the game's mechanics
mechanic:building

Matches any game with Pattern Building, Route/Network Building, or Deck / Pool Building mechanics

publish Matches a part of the name of any of the game's publishers
publish:z-man

Matches any game published by Z-Man Games

design Matches a part of the name of any of the game's designers
design:uwe

Matches any game designed by someone named Uwe

Comparative Terms

All of the terms in this section may use one the following operators to specify conditions:

  • =
  • <
  • >
  • >=
  • <=

Note: A colon (:) may not be used with these terms

Tag Aliases Description Example
rating-votes Compares against the number of user ratings the game has received rating-votes>=1000

Matches any game with at least 1000 user ratings

average-rating Compares against the average user rating of a game average-rating<6.5

Matches any game with an average rating below 6.5

geek-rating Compares against the game's BGG GeekRating metric geek-rating>=8.0

Matches any game with a GeekRating of at least 8.0
